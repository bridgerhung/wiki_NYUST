## 前言
致 親愛的學弟妹，我是111年畢業的108課綱第一屆的學長，希望這篇文章系列所包含的內容能帶給你們一些助益。  

包含將去年的經驗全部濃縮精華，希望可以將經驗傳承  
檔案勾選的眉角，如何避開選填志願的地雷，還有撰寫自述及多元心得，及面試重點，教授問什麼問題，該如何接招，全部收錄在內。

:::note 特別感謝
這篇文章由我過去朝夕相處的老師和同學們，還有面試大學的教授，和一同競爭的夥伴他們和我激盪出火花，還有我的大學同學，才能夠產出這些內容。**感謝他們**
:::
